## 项目基本信息

项目名称：基于 ASP.NET Core MVC 的建筑租赁设备全生命周期管理平台。

项目仓库名：EquipmentRental。

项目类型：B/S 架构 Web 管理系统。

项目目标：面向建筑设备租赁场景，解决传统纸质台账、Excel 管理、线下合同流转、人工进场核验和安全记录难追溯等问题，构建设备从入库到退场的全生命周期数字化管理平台。

## 核心业务流程

```text
设备入库 -> 资质审核 -> 线上调度 -> 进场核验 -> 安全交底 -> 使用监管 -> 退场评价
```

系统不是单纯的设备 CRUD，而是围绕设备租赁全流程设计。设备状态、调度单状态、合同状态、故障状态和退场状态都由业务动作驱动变化。

## 系统模块

系统包含用户与认证、设备台账、资质审核、线上调度、合同管理、进场核验、安全交底、巡检监管、故障处理、退场评价、首页看板、站内消息和文件访问等模块。

## 架构口径

系统采用 ASP.NET Core MVC 分层架构：

- Controller 层负责接收请求、执行授权、处理 ModelState、调用 Service、返回 View/JSON/File。
- Service 层负责业务规则、状态流转、事务边界、审计日志和通知。
- Entity 层对应数据库表结构，只保存属性和导航关系。
- ViewModel 层面向页面输入输出，避免直接暴露实体。
- Razor View 层负责页面展示、表单和交互。
- AppDbContext 负责 EF Core 实体集合、关系、索引和约束配置。
