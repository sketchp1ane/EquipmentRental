# 04 · 进场核验

## 4.1 获取核验码（调度员）

使用账号：**调度员** `demo.dispatcher@equiprental.com` / `Demo@123456`

路径：调度单详情页 → 「核验码」卡片 → 查看 UUID 与二维码

- 页面会展示形如 `c4a3e8d2-... ` 的 UUID
- 同时显示一张二维码图片（由 QRCoder 生成）
- **建议**：复制这段 UUID，稍后在核验页面粘贴

---

## 4.2 执行核验（项目负责人 / 设备管理员）

使用账号：**项目负责人** `demo.projectlead@equiprental.com` / `Demo@123456`
（或设备管理员同样可以核验）

路径：左侧导航 → 「进场核验」→ 输入核验码 (`/Verification`)

**输入字段：**
- 核验码：粘贴 4.1 步得到的 UUID

**系统自动校验：**
1. 设备编号是否存在 → `DEMO2-EQ-001` ✅
2. 证件是否齐全且未过期 → 产品合格证、年检报告、保险 ✅
3. 设备状态是否为"出租中" ✅
4. 核验码是否在有效期（租赁开始日起 3 天内）✅

**预期结果：** 核验通过，记录核验人与核验时间。

> 如果演示当天已超过"租赁开始 +3 天"，可回到调度单重新生成核验码，或将"实际开始日期"调整为今天。
