## 5.2 安全注意事项（富文本，直接复制粘贴到 Summernote 编辑器）

> 编辑器支持粘贴 HTML 与格式化文本，复制下面的整段（含标题加粗）即可。

```html
<p><strong>一、作业环境检查</strong></p>
<ul>
  <li>作业区域须设置安全警戒线与警示标志，严禁无关人员进入。</li>
  <li>地下管线须由技术人员提前排查，挖掘作业须保持安全距离不小于 1 米。</li>
  <li>遇雨、雾、大风等恶劣天气立即停止作业。</li>
</ul>
<p><strong>二、设备操作规程</strong></p>
<ul>
  <li>作业前对液压系统、斗齿、行走机构、制动装置等进行例行检查，发现隐患立即停机。</li>
  <li>严禁在未停稳的情况下下车或进行维修。</li>
  <li>回转半径内严禁站人，司机须鸣笛警示后方可作业。</li>
</ul>
<p><strong>三、个人防护</strong></p>
<ul>
  <li>所有进入施工区域人员必须佩戴安全帽、反光背心、防砸鞋。</li>
  <li>操作室内严禁饮食、吸烟，驾驶员作业期间不得接打手机。</li>
</ul>
<p><strong>四、应急处置</strong></p>
<ul>
  <li>发生设备故障立即停机、切断电源，设置警示标志，通知设备管理员与安全员。</li>
  <li>发生人员伤害立即启动应急预案，第一时间拨打 120 并报告项目负责人。</li>
</ul>
```
