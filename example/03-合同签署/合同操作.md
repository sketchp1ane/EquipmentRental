# 03 · 合同签署

使用账号：**调度员** `demo.dispatcher@equiprental.com` / `Demo@123456`

路径：调度单详情页 → 「查看合同」(`/Contract/{orderId}`)

## 3.1 在线预览合同
点击 **在线预览**，系统渲染 HTML 格式的合同内容（甲乙双方、设备、租期、租金、押金均已自动填充）。

## 3.2 导出 PDF
点击 **导出 PDF**，QuestPDF 生成标准合同 PDF，可下载保存或打印盖章。

## 3.3 上传扫描件（模拟线下盖章后回传）
点击 **上传扫描件**，选择：

- `合同扫描件/合同扫描件.pdf`（优先使用，演示 PDF 上传）
- 或 `合同扫描件/合同扫描件.jpg`

上传成功后合同状态：`草稿` → `待签署` → 点击 **标记为已签署** → `已签署`。
