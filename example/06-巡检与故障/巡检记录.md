# 06.1 · 日常巡检

使用账号：**安全员** `demo.safetyofficer@equiprental.com` / `Demo@123456`
（设备管理员同样可登记）

路径：左侧导航 → 「巡检记录」→ 「新增巡检」(`/Inspection/Create?orderId={id}`)

---

## 巡检记录一（正常）

| 字段 | 值 |
|---|---|
| 关联调度单 | `DEMO2-EQ-001` 的当前调度单 |
| 巡检日期 | 演示当天 +10 天 |
| 整体状态 | 正常 |
| 巡检项目 | 勾选"液压系统"、"发动机"、"行走机构"、"制动装置"、"钢丝绳/吊具"、"电气系统" |
| 备注 | 液压系统压力正常，发动机怠速平稳，各连接件无松动，斗齿磨损在允许范围内。 |
| 现场照片 | `巡检照片/液压系统-正常.jpg`、`巡检照片/斗齿磨损.jpg` |

---

## 巡检记录二（异常，为后续故障工单做铺垫）

| 字段 | 值 |
|---|---|
| 关联调度单 | 同上 |
| 巡检日期 | 演示当天 +14 天 |
| 整体状态 | 异常 |
| 备注 | 主液压泵运行中出现异常噪音，工作油压较前日下降约 15%，疑似密封圈损坏，建议立即停机检修。 |
| 现场照片 | `故障照片/异常噪音.jpg` |

异常巡检记录会出现在设备详情页的巡检历史列表，并提醒跟进。
