# 06.2 · 故障上报 & 工单处理

## 6.2.1 上报故障

使用账号：**项目负责人** `demo.projectlead@equiprental.com` / `Demo@123456`
（安全员也可上报）

路径：调度单详情页 → 「故障上报」(`/Fault/Create?orderId={id}`)

| 字段 | 值 |
|---|---|
| 故障描述 | 主液压泵出现异常噪音，工作油压下降明显，主油管附近出现渗油现象，已停止一切挖掘作业，设备停机等待维修。 |
| 严重程度 | 严重 |
| 发生时间 | 演示当天 +14 天 上午 10:30 |
| 现场照片 | `故障照片/油管渗漏.jpg`、`故障照片/异常噪音.jpg` |

提交后：
- 故障工单状态：**待处理**
- 设备状态自动变为：**维修中**
- 设备管理员会收到站内消息通知

---

## 6.2.2 接单 & 处理

切换账号：**设备管理员** `demo.deviceadmin@equiprental.com` / `Demo@123456`

路径：左侧导航 → 「故障工单」(`/Fault`)

1. 点击刚收到的工单 → 「接受工单」 → 状态变 **处理中**
2. 现场检修完成后 → 「关闭工单」填写：

| 字段 | 值 |
|---|---|
| 处理结果 | 更换主液压泵密封圈及渗漏油管，补充液压油至正常液位，空载试运行 30 分钟油压稳定，已恢复使用。 |
| 维修费用(元) | 3800 |

关闭后：
- 工单状态：**已关闭**
- 设备状态恢复为 **出租中**（或空闲，视业务规则）
- 项目负责人收到通知
