# 07 · 退场申请 & 评价

## 7.1 提交退场申请

使用账号：**项目负责人** `demo.projectlead@equiprental.com` / `Demo@123456`

路径：调度单详情页 → 「申请退场」(`/Return/{orderId}/Apply`)

| 字段 | 值 |
|---|---|
| 实际退场日期 | 演示当天 +32 天（或接近租期结束日） |
| 设备总体状况描述 | 设备在使用期间运转正常，中途出现一次液压故障已及时修复，其余无碰撞损伤。退场前已完成外观清洁与油箱检查，油箱液位与进场一致，归还现场已拍照存档。 |

提交后状态：**待评价**，设备管理员收到通知。

---

## 7.2 填写退场评价

切换账号：**设备管理员** `demo.deviceadmin@equiprental.com` / `Demo@123456`

路径：左侧导航 → 「退场管理」→ 点击刚才的申请 → 「填写评价」(`/Return/{id}/Evaluate`)

| 字段 | 值 |
|---|---|
| 设备外观评分 | 4（有轻微划痕但无结构性损伤） |
| 功能完好性评分 | 4（液压故障已修复，功能正常） |
| 损耗描述 | 右侧履带轻微磨损，斗齿磨损在允许范围内，液压管路已于维修时更换。整体状态可接受。 |
| 损耗扣款(元) | 500 |
| 备注 | 扣款主要用于下次进场前的例行维护与斗齿更换准备。 |

系统自动计算：

```
押金 8000 − 扣款 500 = 退还金额 7500 元
```

提交后：
- 退场申请状态：**已完成**
- 设备状态：**空闲**（可重新投入新一轮调度）
- 项目负责人收到通知
