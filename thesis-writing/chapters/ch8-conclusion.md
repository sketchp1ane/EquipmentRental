# 8 结论

本文围绕建筑租赁设备管理业务，设计并实现了一个基于 ASP.NET Core MVC 的建筑租赁设备全生命周期管理平台。系统以设备为核心，以调度单串联租赁过程，覆盖设备入库、资质审核、线上调度、合同管理、进场核验、安全交底、使用监管、故障处理和退场评价等环节。系统采用 B/S 架构和 MVC 分层设计，将请求处理、业务规则、数据实体、页面模型和页面展示进行职责划分，提高了系统结构清晰度和可维护性。

在功能实现方面，系统实现了用户认证与角色权限控制、设备台账管理、资质审核、调度合同、进场核验、安全交底、巡检故障、退场评价、首页看板、站内消息和文件安全访问等模块。通过状态字段和服务层业务规则，系统能够约束设备和调度单在不同业务阶段的流转，避免未审核设备参与调度、未签署合同直接核验等问题。通过文件上传校验、CSRF 防护、富文本过滤和受控下载，系统具备一定的安全防护能力。

系统测试结果表明，平台能够满足毕业设计场景下建筑租赁设备管理的主要业务需求，关键功能和状态流转基本符合预期。但由于时间和条件限制，系统仍存在不足，例如尚未接入移动端 App、真实硬件扫码设备、具有法律认证效力的在线电子签章、GPS 实时定位、第三方 ERP 系统和财务发票系统。后续可以在移动端适配、智能调度、数据统计分析、电子签章认证和企业系统集成等方面继续扩展，以进一步提升系统的实用性和业务覆盖范围。
